# VS-site-v1
Vachi Services Website

## Quick Deploy

### Option 1: Vercel (Recommended)
```bash
# Install Vercel CLI
npm install -g vercel

# Deploy to Vercel
vercel --prod
```

## Project Structure
- `project/` - Main React + Vite application
- `website/project/` - Alternative deployment configuration

## Development
```bash
cd project
npm install
npm run dev
```

## Build
```bash
cd project
npm run build
```
